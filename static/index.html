WIZARDS ON DUTY

THE WIZARD FACTORY

Every Claude Code session (and every subagent it summons) appears here as a wizard. Nothing registers itself — the server tails the transcripts Claude Code already writes to ~/.claude/projects/ and infers what each agent is up to.

READING THE ROOM

  • Cauldron — running shell spells (Bash)
  • Bookshelves — reading & searching files (Read / Grep / Glob)
  • Writing desks — editing & writing code (Edit / Write)
  • Crystal ball — the web & MCP far-realms (WebFetch / mcp tools)
  • Summoning circle — spawning subagents (Task / Agent / Skill)
  • Petition board — plans, questions, and ! = needs your approval
  • Cafe — turn finished, awaiting your next prompt with a fresh order from Earl Grey
  • Hearth armchairs — idle 15min+, dozing off
  • Apprentices (smaller, beardless) are subagents; they vanish when their task ends.
  • Hooded wizards with a codex chip are OpenAI Codex CLI sessions (read from ~/.codex/sessions); one-shot codex exec runs finish and leave rather than waiting around.

SHARPER SIGNALS (OPTIONAL)

Polling sees everything within ~1s, but can't always tell a long-running spell from a permission prompt. For instant "needs your blessing" alerts:

python3 server.py --install-hooks  (and --uninstall-hooks to undo; a backup of settings.json is kept, applies to new sessions)

DEMO

python3 server.py --demo --port 7778 fills the tower with fake wizards.
